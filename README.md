Name of Game: E.M’S Rush

Description:
Help Lovely run through the roads of EM’s Barrio as she races against the clock to get to school on time. 
Control Lovely and make her run fast to avoid nasty poop, swampy mud, filthy dogs and other obstacles along the way. 
Buy power-ups such as Time freeze, Padyak Booster, and a lot more to help her reach school faster. 
Run as fast as you can, but don't forget to collect coins!

Genre: Action-Adventure

Platform: Cross Platform

Language: Python

Tools: Pygame
